CLI 환경에서 Git Clone 후 STS에서 프로젝트 import 하기

clone 한 프로젝트 상위 폴더에서 STS 실행 -> [Package Explorer] 우클릭 -> [import] -> [General]-[Projects from Folder or Archive] -> 프로젝트 폴더 선택
